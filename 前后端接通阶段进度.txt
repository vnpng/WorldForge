### WorldForge 前后端接通阶段进度报告 (更新日期: 2026-03-12)

#### ✅ 已接通功能 (已实现云端同步与持久化)
1.  **账号体系**：
    *   用户登录、注册、邀请码校验、JWT 鉴权拦截。
2.  **会话管理 (Sessions)**：
    *   会话列表加载（支持 RPG/Chat 视觉区分）。
    *   新建对话与新建开局（支持字符串 ID 与即时存档）。
    *   会话重命名（实时同步）。
    *   物理删除会话（云端同步销毁）。
    *   对话记录导出（JSON 格式下载）。
3.  **资产系统 (Assets)**：
    *   世界、角色、引擎预设的全量 CRUD 与 Sortable.js 拖拽排序同步。
4.  **AI 聊天引擎**：
    *   流式代理转发、流式/非流式切换逻辑、思维链 (CoT) 与推荐行动解析。
5.  **配置偏好**：
    *   API 节点管理（增删改、批量导入）、高级调节参数的本地持久化记忆。

#### ❌ 逻辑断路/已知 Bug (需优先修复)
1.  **置顶同步 Bug**：前端 `syncSession` 尚未包含 `is_pinned` 字段，导致发送消息后置顶状态会丢失。
2.  **详情页开局断路**：发现页详情里的“开启冒险”按钮目前无法生成真实会话，仅为视图切换。
3.  **资产公开逻辑**：后端 `Worlds` 和 `Characters` 表暂无 `is_public` 字段，无法支持资产共享。

#### ⏳ 待接通/开发中功能
1.  **发现页面 (Discover)**：目前仍为 Mock 数据，需开发 `/api/discover` 接口及前端适配。
2.  **数据管理中心**：个人中心里的“全量数据导出/导入/重置”功能尚未编写逻辑。
3.  **角色状态动态同步**：状态面板（HP/MP 等）与背包目前为 Mock 数据，需实现 AI 提取与数据库字段绑定。
4.  **创作者中心**：目前仅为 UI 占位。
5.  **高级扩展能力**：白皮书提到的 Lorebook 注入、上下文压缩 (STM) 逻辑尚未在后端实施。
